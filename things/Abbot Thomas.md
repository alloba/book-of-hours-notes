# Abbot Thomas

## Description
'Abbot Thomas'

Thomas was one of the first abbots here, at the beginning of the [[Solar Gothic period]]. This bust is much more recent - a nineteenth-century bronze. An inscription reads: SPEECH, THE FIRST SWORD.
